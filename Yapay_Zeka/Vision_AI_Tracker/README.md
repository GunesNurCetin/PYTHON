Vision AI Tracker (Yapay Zeka Destekli Görüntü Takipçisi)
A real-time face detection and video processing application using OpenCV. It can track human faces and add dynamic security timestamps to the live feed.
OpenCV kullanarak geliştirilmiş, gerçek zamanlı yüz tespiti ve video işleme uygulamasıdır. İnsan yüzlerini takip edebilir ve canlı yayına dinamik güvenlik zaman damgaları ekleyebilir.

🚀 Features (Özellikler)
Face Detection: Detects multiple faces in real-time. (Eş zamanlı olarak birden fazla yüzü tespit eder.)

Live Timestamping: Embedded security-style date/time overlay. (Güvenlik kamerası tarzında gömülü tarih/saat katmanı.)

Gray-Scale Processing: Optimized for high FPS performance. (Yüksek FPS performansı için gri tonlamalı optimizasyon.)

🛠 Tech Stack (Kullanılan Teknolojiler)
Language: Python 3.x

Library: OpenCV (Open Source Computer Vision Library)

Model: Haar Cascade Classifiers


pip install opencv-python
python vision_engine.py
